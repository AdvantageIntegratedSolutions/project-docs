Document New Project Nothing new to learn. The folder organization will do most of the work.

All documentation is stored under the "/docs" folder within the project-docs Angular QuickStart application. Each documented project has a folder titled after the project name making it clear and easy to organize documentation. Inside each folder is an index.doc.html file and a 'sections' folder that contains each section of documentation.

Steps to document a new project:
  1. Add a new folder under "/docs", titling it with the name of your project.
  2. Add the name of the new document to the "_docs.js" file.
  3. Add the new project to the project-docs list of current projects("project-docs.doc.html")
