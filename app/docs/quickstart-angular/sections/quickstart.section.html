Quickstart Starting a new project.

Once your environment is all setup, starting a new project is easy. The quickstart-angular framework is hosted on GitHub and no external dependencies are required. This is ideal, because we will clone this repo, quickly adapt to our new project and hit the ground running.

Step-by-step instructions:
  1. Create new repo on Github and title the repo to the name of your project.
  2. Clone the quickstart-angular repo and provide the name of your new project. git clone https://github.com/AdvantageIntegratedSolutions/quickstart-angular.git new-app-name
  3. Update the git origin remote with your app's repo url. git remote set-url origin new-repo-git-url
  4. Install node modules at the root of your new project folder. npm install
  5. Configure the 'config/app.config.js' file to your new project.
  6. Start your local server on 'localhost:3000'. gulp local
  7. You are all setup. Visit 'localhost:3000' to view your project.
