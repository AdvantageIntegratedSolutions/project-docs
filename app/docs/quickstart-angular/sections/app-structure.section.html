App Structure

quickstart-angular is an angular web-framework to build consistent, quality QuickBase and ServiceNow single-page applications.

quickstart-angular allows developers to fire up single-page application within minutes. Reusable angular modules can be quickly imported from a centralized angular module library.

It allows developers to spend less time setting up a suitable environment and more time developing client solutions. The framework includes a intuitive local development environment, a consistent directory structure across projects and common helper tasks to compile, serve and deploy your project.
