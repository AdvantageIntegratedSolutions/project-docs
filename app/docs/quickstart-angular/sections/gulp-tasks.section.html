gulp Tasks Automated tedious tasks.

gulp is responsible for all of the automation within quickstart-angular. You can safety assume that any actions taken to your project are somehow related to gulp. All gulp tasks are loaded in our gulpfile.js at the root of the project. All tasks are located in our /build folder.

List of tasks
Task Purpose
local launches local server at localhost:3000
deploy deploy all application source code to QuickBase
js-dev run all javascript tasks required for development
js-prod run all javascript tasks required for production
css-dev run all css tasks required for development
css-prod run all css tasks required for production
html-dev run all html tasks required for development
html-prod run all html tasks required for production
upload-html upload dist index html to QuickBase application
upload-assets upload dist assets to QuickBase application
templates load all html files into template cache
init clear and populate readme
Any of these tasks can be run individually by running 'gulp task-name' within your command line however the majority of these are automatically bundled into the tasks 'local' or 'deploy'.
