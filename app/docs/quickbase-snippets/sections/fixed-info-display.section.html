fixed-info-display Display a fixed info display on quickbase form to improve visibility of important data points. Instructions
  1. Create a new 'Formula - Text' field and title it 'fixed-info-content-image'.
  2. Enable 'Allow some HTML tags'.
  3. Add this field to bottom of QuickBase form.
  4. Copy this snippet into the formula field.
  5. Create 2 new 'Formula - Text' fields, 'fixed-info-content' & 'fixed-info-color'.
  6. Add both fields to bottom of QuickBase form.
  7. Update the 'fixed-info-content' field with data that you want displayed.
  8. Update the 'fixed-info-color' field with the hexidecimal(#FF0000, #EEEEEE, etc) color desired for the background color of the display.
  9. Done.
