css-form-tabs Add CSS customizable tabs on a QuickBase form and toggle multiple forms.
  1. Create a 'Formula - Text' field and title 'Tabs 1'.
  2. Enable 'Allow some HTML tags to be inserted in the field'.
  3. Copy this snippet into the formula field.
  4. Create a new form for each additional tab needed.
  5. Update the 'tab settings' form ids in the formula to match the new form ids(dfids).
  6. Duplicate the 'Tabs 1' formula field for each additional tab needed (i.e. Tab 2, Tab 3, etc)
  7. Update the 'activeTab' variable in each new formula field to the tabs position.(i.e. 1, 2, 3)
  8. Add one tab field to each form.
