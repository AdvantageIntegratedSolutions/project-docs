do-save-add Add an additional Save button on a form and optionally execute form actions.
  1. Create a new 'Formula - Text' field.
  2. Enable 'Allow some HTML tags'.
  3. Copy this snippet into the formula field.
  4. Insert optional custom jQuery actions prior to DoSaveAdd() call.
    
      //example 1
      onclick='$(\"#_fid_33\").prop(\"checked\", true); DoSaveAdd()';

      //example 2
      onclick='$(\"#_fid_21\").val("Active"); DoSaveAdd()';

      //example 3
      onclick='$(\"#_fid_33\").prop(\"checked\", true); $(\"#_fid_21\").val("Active"); DoSaveAdd()';
