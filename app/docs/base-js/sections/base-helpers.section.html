BaseHelpers

BaseHelpers is an additional light-weight object with commonly used functions that is bundled with base-js. It is automatically included when base-js is loaded.

Function Purpose Parameters
setCookie Set value of a browser cookie. (name, value, hours)
getCookie Retrieve a value of a browser cookie. (name)
inverseTables Return an object with reversed key-values. (key-value object)
getUrlParam Retrieve value of URL parameter (name)
formatDateElement Format date property(hour, minute) to a two-digit #. (value)
dateToString Format date to mm-dd-yyyy (milliseconds)
dateTimeToString Format date to mm-dd-yyyy HH:MM (milliseconds, timezone)
durationToString Format duration value (milliseconds, format)
timeOfDayToString Format time of day value. (milliseconds)
redirectToEditForm Redirect to QuickBase table edit form. (dbid, rid, root)
redirectToViewForm Redirect to QuickBase table view form. (dbid, rid, root)
downloadFile Download QuickBase file attachment. (dbid, rid, fid, version)
BaseHelpers is bundled and automatically loaded with base-js. There is no additional requiring to access these functions.
