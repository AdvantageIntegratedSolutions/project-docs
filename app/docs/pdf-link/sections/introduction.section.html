Introduction

pdf-link is a tool to generate and/or store PDF's. PDF's are generated by converting HTML/JavaScript pages.

pdf-link utilizes a third-party PDF converter called html2pdfrocket to convert HTML pages to PDF documents. These generated pdf documents can be downloaded or saved back to QuickBase. This tool is entirely configured using URL query parameters.

pdf-link is a simple rails application hosted on the Advantage web server that can be pinged from anywhere. It is offered to our clients at $1,188 a year for unlimited pdf conversions.

The endpoint for pdf-link is https://pdflink.advantagesoftware.net/convert/new
