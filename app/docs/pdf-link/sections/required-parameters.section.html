Required Parameters

The following parameters are required for converting an HTML page to PDF.

Required Parameters Purpose
url Url of HTML page being converted. Must be url encoded.
rocket Must be set to true. Tells pdf-link to use newer pdf converting service.
FileName Name of desired output file. Exclude file extension. 
	
	const url = encodeURIComponent("https://www.google.com");
	const filename = "invoice-10001"

	const pdfUrl = "https://pdflink.advantagesoftware.net/convert/new?rocket=true&" + "url=" + url + "&FileName=" + filename;

	$.get(pdfUrl);
The 'rocket' parameter forces pdf-link to use html2pdfrocket intead of ConvertAPI. Convert API is a previous third-party service that had issues splitting tables and hard to reach support. Always remember your first url query parameter will follow a '?' not a '&'.
