Quickstart Let's convert a simple HTML page.

To convert an HTML page to a PDF, we will do a simple GET on our pdf-link endpoint while passing all neccessary query parameters.

Try opening the following url into your browser to see a simple example in action.

https://pdflink.advantagesoftware.net/convert/new?FileName=timecard&url=http%3A%2F%2Fwww.advantagequickbase.com%2F&rocket=true&serve=true

Notice we are only passing a the minimum required url parameters to our pdf-link endpoint. Additional functionality can be utilized using other query parameters documented below.
