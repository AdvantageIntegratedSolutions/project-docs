Headers and Footers

Headers and footers can be configured using the above format parameters. Simple values such as page numbers or date stamps can be easily passed across in the url and the converter will auto-include on these values on every page.

Complex Headers/Footers

Often times you may need to have headers and footers that are repeated on every page but are much more complex than simple page numbers or a datestamp. To accomplish this you can utilize the 'HeaderUrl' and 'FooterUrl' parameters.

You can pass in an additional encoded url to have the entire content of a page be embedded as a header or footer.

	
	//required parameters
	const url = encodeURIComponent("https://www.google.com");
	const filename = "invoice-10001"

	//parameters to set header/footer with html content
	const headerUrl = encodeURIComponent("https://ais.quickbase.com/header.html")
	const footerUrl = encodeURIComponent("https://ais.quickbase.com/footer.html")

	const parameters = 
		"&url=" + url + 
		"&FileName=" + filename + 
		"&headerUrl=" + headerUrl + 
		"&footerUrl" + footerUrl 

	const pdfUrl = "https://pdflink.advantagesoftware.net/convert/new?rocket=true" + parameters;

	$.get(pdfUrl);
Do not attempt to use CSS page-break properties or javascript algorithms to solve the fixed repeated footer on every page problem. Many have tried, many have failed.
