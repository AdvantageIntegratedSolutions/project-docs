Saving PDF Save the pdf back to QuickBase

Many times you may need to convert a QuickBase page and save the PDF back to QuickBase.

Let's say you have a button on a customer form titled "Generate Invoice". The user could click this button, generate an invoice by converting an invoice custom page to a PDF and then save this new invoice pdf back to the customer record.

There are three options to save a PDF:

  • Attach the PDF to an existing record in a file attachment field in QuickBase.
  • Create a child record and relate to a parent record in QuickBase.
  • Download the PDF to local clients computer.

Each use case uses different url parameters. Below are the different required parameters to save a pdf to QuickBase.

Attach to record
Parameter Value
domain quickbase domain of application
app_token application's app token, required if enabled in application
dbid dbid of table that contains record
rid record id
file_fid field id of file attachment field 
	
	//required parameters
	const url = encodeURIComponent("https://www.google.com");
	const filename = "invoice-10001"

	//parameters to save pdf to record
	const domain = "ais"
	const dbid = "bfghighi";
	const rid = 10001;
	const file_fid = 10;

	const parameters = 
		"&url=" + url + 
		"&FileName=" + filename + 
		"&domain=" + domain + 
		"&dbid" + dbid + 
		"&rid" + rid + 
		"&file_fid" + file_fid;

	const pdfUrl = "https://pdflink.advantagesoftware.net/convert/new?rocket=true" + parameters;

	$.get(pdfUrl);
Create new record
Parameter Value
domain quickbase domain of application
app_token application's app token, required if enabled in application
rid record id of parent record
parent_fid field id of parent reference field
child_dbid dbid of table where new record will be saved with pdf
file_fid field id of file attachment field in child table 
	
	//required parameters
	const url = encodeURIComponent("https://www.google.com");
	const filename = "invoice-10001"

	//parameters to save pdf to record
	const domain = "ais"
	const rid = 10001;
	const parent_fid = 6;
	const child_dbid = "bfghiag3"
	const file_fid = 10;

	const parameters = 
		"&url=" + url + 
		"&FileName=" + filename + 
		"&domain=" + domain + 
		"&rid" + rid + 
		"&parent_fid" + parent_fid +
		"&child_dbid" + child_dbid + 
		"&file_fid" + file_fid;

	const pdfUrl = "https://pdflink.advantagesoftware.net/convert/new?rocket=true" + parameters;

	$.get(pdfUrl);
Download

If you only want to download the pdf, ignore the above parameters and just include the required and optional format parameters.
