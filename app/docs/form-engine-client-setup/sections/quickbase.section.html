Quickbase Actions required to be performed on quickbase
Copy Application
  • Go to Settings under the application name
  • Go to App Management
  • Copy app
    • Data: select copy this application WITHOUT data
    • Users & Roles: Copy users in their roles
Get Table DBID's
  • Go to Settings under the application name
  • Go to App Management
  • Under Show App Info list go to Show Support Information
    • This will contain all DBID's that will need to be mapped for the clients version of the application.
