Sites

Each asset rendered on the calendar belongs to a Program/Site in the MCR Tracker application. Each program/site is highlighted using a unique color and displayed in the color legend.

Assigning a Color

Each site is highlighted on the calendar as a different color and can be configured from the 'Programs' table in QuickBase.

Updating/Adding color to Program/Site:
  1. From the MCR Tracker application, navigate to 'Programs'.
  2. Open a program record in edit mode.
  3. Under the Admin section, select a color for the 'Calendar V2 Background Color' dropdown.
Any program that is loaded in the source data but is not assigned a color in the 'Programs' table will be highlighted using a default 'gray' value. Configuring Colors

The list of available site legend colors can be configured in QuickBase by updating the 'Calendar V2 Background Color' choices and then the associated 'Calendar V2 Background Color HEX' formula.
