Introduction

MCR Asset Calendar is a JavaScript/Angular calendar for Coke's MCR program. It allows users to view & filter MCR assets in an Outlook-inspired calendar allowing monthly/weekly/daily and list views. It is driven using an external JavaScript library, FullCalendar.io. The project is hosted within the MCR Tracker QuickBase application.

This project is a replacement for an old gnatt-chart developed by previous Advantage & Coke teams. The previous project relied on Tree-Grid service.
