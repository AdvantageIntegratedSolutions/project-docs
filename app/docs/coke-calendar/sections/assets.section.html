Assets

All calendar events on the calendar represent an asset from the MCR Tracker application. The calendar queries the 'Assets' table in MCR Tracker QuickBase application and renders all assets.

Source Data

The calendar loads all assets that are found in the native quickbase report, Report #152. Any filter changes to this report will automatically be relected on the calendar. Columns in this report do not control any calendar functionality and can be updated without any impact.

The calendar uses QuickBase fields 'Start Date' and 'End Date' to determine the calendar start-end date of the calendar event.

Event Title

When an calendar event is rendered on the calendar, the event displays a title. The calendar event title can be configured in QuickBase by updating the 'Calendar V2 Event Title' formula.

Hover Data

When an calendar event is hovered, a hover-text box will be displayed providing additional asset information. The hover content can be configured in QuickBase by updating the 'Calendar V2 Hover Content' formula.
