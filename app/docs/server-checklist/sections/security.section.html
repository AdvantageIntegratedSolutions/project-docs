Step 4: Security

Security management is complicated but some high-level maintainance can make a world of difference and keep our servers secure and safe.

When logging onto server, the server will tell you if you have any packages and security updates to install. run 'sudo apt-get update' if neccessary. Navigate to the commonly used areas of the servers and ensure no folders are open to the world with write or execute permissions. Check active connections and look for suspicious IP addressses from places like China or Russia. 'netstat -natp'
