Automated Webhooks

Follow these steps to setup an automated webhook using Workato.

Step 1: Trigger
  • Login to Workato account using credentials in developer credentials document.
  • Click "Create a new recipe" in upper right hand corner.
  • Select "Scheduler" from "Application" dropdown.
  • Select "New scheduled event" from "Trigger" dropdown.
  • Select desired time interval option from "Every" dropdown.
  • Click "Next".
Step 2: Actions
  • Select "HTTP" from "Application" dropdown.
  • Select "POST" from "Method" dropdown.
  • Insert the isomorph Endpoint URL in the "Request URL" input field.
  • Click "Add/remove optional fields" and select "Request Header" and "Request Body".
  • Insert the isomorph required params and content headers into the "Request Headers" input field. (see below)
  • Insert {} into the "Request Body" input field.
  • Click "Next".
    
      Request Headers:

      Content-Type: application/json
      Accept: application/json
      realm: 
      dbid: 
      pageID:
      user:

      Request Body:
      {}
Don't forget to add the 'Content-Type' and 'Accept' request headers. Step 3: Connections
  • Click connection and select "No Auth Header" from existing connections dropdown.
  • Click "Next".
Step 4: Description
  • Name your recipe using following naming convention: "Automated Webhook - client name - webhook name"
  • Select "Private" in "Access" dropdown.
  • Click "Finish"
  • Click arrow next to "Test recipe" and select "Start Recipe"
  • Done.
