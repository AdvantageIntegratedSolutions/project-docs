Quickstart

The specifications listed below are for a typical webhook. A new hook can be created under the "Webhooks" section in QuickBase's table settings.

Endpoint URL: https://d48pr0dwh3.execute-api.us-east-1.amazonaws.com/prod/LoadPage

HTTP method: POST

Message format: JSON

Message body: This input takes a stringified object (JSON) that will be converted to GET parameters when making a request to your code page. See example: {"notes":"This data was posted by a client side page via a webhook. Neat."}

In addition to the above, there are a number of required parameters that you must pass as header values to the aforementioned endpoint. In QuickBase these are defined under the Message header section.

isomorph-js uses PhantomJS to execute your code. Due to this, you may run into trouble with certain ES6 features such as classes, arrow functions and various iterators. Keep your code as vanilla as possible and it's unlikely you will have any problems.
