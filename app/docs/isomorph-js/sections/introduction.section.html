Introduction

isomorph-js is a tool that enables execution of client-side JavaScript code on the server.

Instead of having to endure a lengthy deployment and testing process for a simple script in Rails, isomorph-js gives you the option to keep all of your code in QuickBase.

The most common use case for isomorph-js at the time of this writing is for webhooks, but there are plans to develop a scheduling feature in the near future.
