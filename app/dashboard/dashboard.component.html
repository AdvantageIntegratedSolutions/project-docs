Welcome This is the Project Documentation Library. It is a platform to develop and organize documentation for common libraries, projects and tools.

project-docs is the applicationi that drives the standard UI to improve ease of use and readability, reusable components to shorten the documentation process and help construct documentation standards.

All documentation is written in common HTML & SASS, no new rules or markdown to learn.

Functionality includes reusable components, responsive CSS framework, automatic side-navigation/routing and syntax highlighted code examples.

Inspired by other more robust documentation UI's:

  • Stripe API
  • AngularJS
  • Underscore

This is just the project library dashboard, click the project-docs project to see how to document a new project.
