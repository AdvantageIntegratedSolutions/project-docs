Welcome

project-docs is a platform to develop and store documentation for common libraries, projects and tools.

project-docs provides a standard UI to improve ease of use and readability, reusable components to shorten the documentation process and help construct documentation standards.

All documentation is written in common HTML & SASS, no new rules or markdown to learn.

Functionality includes reusable components, responsive CSS framework, automatic side-navigation/routing and syntax highlighted code examples.

Inspired by other more robust documentation UI's:

  • Stripe API
  • AngularJS
  • Underscore
All project documentation can be found at http://docs.advantagequickstart.com/
