Welcome

The Project Documentation Library (project-docs) is a platform to develop and organize documentation for common libraries, projects and tools.

project-docs provides a standard UI to improve ease-of-use and readability of documentation, reusable components to shorten the documentation process and a centralized library to drive documentation standards.

Inspired by other awesome documentation UI's like Stripe API, AngularJS, Underscore.

This is the project documentation library dashboard, click the project-docs project to see how to document a new project.
